

# 🚀 Nexora - AI Marketing Automation Platform



## 🌟 Introduction
Nexora is an AI-powered marketing automation platform that transforms your marketing workflow in seconds. Built with modern technologies, it offers a sleek interface for managing marketing campaigns, content generation, and performance analytics.




## 💻 Tech Stack

- **Next.js 15** – React framework for production
- **TailwindCSS** – Utility-first CSS framework
- **Shadcn UI** – Reusable component system
- **Framer Motion** – Smooth animations
- **TypeScript** – Type-safe code
- **Number Flow** – Smooth number animations

## 🛠️ Installation

1. Clone the repository:
    ```bash
    git clone 
    ```

2. Install dependencies:
    ```bash
    pnpm install
    # or
    yarn install
    ```

3. Run the development server:
    ```bash
    pnpm run dev
    # or
    yarn dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🔧 Environment Variables

Create a `.env` file in the root directory:

```env
NEXT_PUBLIC_APP_URL=your_app_url
```
